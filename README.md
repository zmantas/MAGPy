# MAGPy

This code is the Python3 translation of the MAGMA code. Originally written in Fotran77 by Fegley and Cameron (1987), it is multicomponent gasmelt chemical equilibrium code based on silicate melt solution models developed by Hastie and coworkers (Hastie and Bonnel, 1985; Hastie et al., 1984; Hastie et al., 1982a, Hastie et al., 1982b). 

The original MAGMA code is available in the `legacy\_fortran\_code` directory. For detailed descriptions see: 

B. Fegley, Jr. & A.G.W. Cameron (1987) A Vaporization Model for Iron/Silicate Fractionation in the Mercury Protoplanet. Earth Planet. Sci. Lett. 82, 207-222.

L. Schaefer & B. Fegley, Jr. (2004)A Thermodynamic Model of High Temperature Lava Vaporization on Io. Icarus 169, 216-241.
